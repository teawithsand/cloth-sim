# cloth-sim

Simple cloth simulation using verlett integration.

It gives you control over:
- Corners
- Gravity
- Spring strength
- Camera

And more!

![Screenshot](./ss1.png)

# Compiling

1. Clone [vcpkg](https://github.com/microsoft/vcpkg)
2. Run `make-build.sh` script
3. Go to `build` directory and run `make`
